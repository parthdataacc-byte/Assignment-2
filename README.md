# Assignment-2
# Part C: Practical Webpage Task
Create a complete HTML page using semantic tags, forms, tables, and text formatting.
Your webpage theme: "My Learning Portfolio"
# 1. Boilerplate Structure
Your file must contain:
<!DOCTYPE html>
<html>
<head> with <title>
<body>

# 2. Semantic Layout (Mandatory)
Use the following structure:
<header> → Page title using <h1> and a short description using <p>
<main> → All main content
At least two <section> elements
Each section must contain one <article>
<aside> → Side note about you or your course
<footer> → Copyright or your name

# 3. Content Requirements
A. Headings & Text
Use at least 3 different heading levels (<h1>–<h6>)
Add paragraphs (<p>) for explanation
Use <br> at least once
B. Links & Images
Add one hyperlink (<a>) to any website (e.g., GitHub, school site)
Add one image (<img>) with proper description
C. Text Formatting
Use:
<sub> (example: H₂O)
<sup> (example: x²)
<code> to display any HTML tag
<details> with a summary and hidden text

# 4. Table Section
Create a table showing "My Subjects" with at least:
3 columns (e.g., Subject, Teacher, Interest Level)
3 rows of data

# 5. Form Section
Create a form with the following fields:
Name (text input)
Email (email input)
Age (number input)
Feedback (text input or textarea)
Submit button
Use headings or paragraphs to label each input clearly.

# 6. Inline vs Block Demonstration
Add a section titled "Inline vs Block Elements" where you:
Write one paragraph explaining the difference
Show at least:
Two inline elements (e.g., <a>, <sup>, <sub>, <code>)
Two block elements (e.g., <p>, <div>, <section>, <article>)
